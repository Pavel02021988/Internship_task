Задание 2. ML

Решение основано на обучающем уроке переноса обучения документации Pytorch, 
была использована предобученная сверточная нейронная сеть resnet34.
Изображения были приведены к размеру 224 х 224 и трансформированы в тензоры
со стандартной нормализацией необходимой для предобученных моделей.
После загрузки модели resnet34 изменен полносвязный слой
под нашу задачу классификации изображений мужчин и женщин.
Оптимизатор Adam, обучающий шаг 0.003, батч 128.
Обучение проходило в две эпохи, метрика accuracy - 89.5
